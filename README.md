The documents declared in this repository are:

- Facebook
- Instagram
- Line
- LinkedIn
- Parler
- Pinterest
- Quora
- Reddit
- Snapchat
- Spotify
- Telegram
- TikTok
- Tumblr
- Twitch
- Twitter
- WeChat
- WhatsApp
- YouTube
